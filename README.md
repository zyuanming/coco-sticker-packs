# Coco Sticker Packs

Open sticker packs for the Coco iMessage app.

## Packs

- `pets-goofy-v1`: original goofy pet stickers for testing remote downloads.
- `cat-meme-v2`: no-text expressive cat face stickers inspired by reaction memes.
