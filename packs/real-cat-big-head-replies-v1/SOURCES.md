# Big-Head Cat Replies Sources

The `real-cat-big-head-replies-v1` pack is built from openly licensed Wikimedia Commons cat photos. Each source image is cropped into a large-head chat sticker, combined with short Chinese reaction text, and exported as a PNG with a transparent canvas, white outline, and soft shadow.
