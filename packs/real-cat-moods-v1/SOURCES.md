# Real Cat Moods Sources

The `real-cat-moods-v1` pack is built from openly licensed Wikimedia Commons cat photos. Each source image is cropped into a rounded sticker card, combined with Chinese reaction text, and exported as a PNG.

- Cropped each source to a square sticker composition.
- Added rounded framing, white outline, soft shadow, and short Chinese reaction text.
- Exported as optimized PNG files for Coco remote sticker delivery.
